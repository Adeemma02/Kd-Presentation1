Introduction

  • Understanding web foundations is key to every modern website.
  • HTML defines the structure.
  • CSS adds design and layout.
  • Together, they form the building blocks of the web.

What is HTML?

HTML stands for HyperText Markup Language.

  • It describes the structure of a webpage.
  • Uses ‚Äútags‚Äù to represent content elements.
  • Example:

<h1>Hello World</h1>
<p>Welcome to my first webpage.</p>

Basic HTML Structure


<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>
  • <head> ‚Äì holds metadata and title
  • <body> ‚Äì contains visible content

Common HTML Elements

  • Headings: <h1> to <h6>
  • Paragraphs: <p>
  • Links: <a href="#">Link</a>
  • Images: <img src="image.jpg" alt="desc">
  • Lists: <ul> and <ol>

What is CSS?

CSS stands for Cascading Style Sheets.

  • It adds style, colors, and layout to web pages.
  • Separates design from structure.

body {
  background-color: #fafafa;
  color: #333;
  font-family: Arial, sans-serif;
}

Ways to Add CSS

1. Inline CSS


<h1 style="color:blue;">Hello</h1>

2. Internal CSS


<style>
  p { color: green; }
</style>

3. External CSS


<link rel="stylesheet" href="styles.css">

CSS Selectors and Properties

  • Selectors target HTML elements.
  • Properties define how they look.

h1 { color: blue; }
.highlight { background: yellow; }
#main { margin: 20px; }

The CSS Box Model

  • Every element is a rectangular box.
  • Composed of: Content ‚Üí Padding ‚Üí Border ‚Üí Margin

Modern Layouts: Flexbox & Grid

Grid Example


.grid {

Best Practices

  • Keep your code clean and indented.
  • Use semantic HTML tags for clarity.
  • Separate structure (HTML) from style (CSS).
  • Use external stylesheets for maintainability.
  • Test across multiple browsers and devices.
